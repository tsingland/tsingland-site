### emacs 常用命令

    #### 移动命令

    	 C + p 向上移动

C + b 向左移动         C + f 向右移动

	 C + n 向下移动

    #### 关闭 emacs

    	 C + x C + c

    #### 打开文件夹

    	 C + x C + f  +entern

    #### 选中文字

    	 C + spaceback

    #### 剪贴文字

    	 C + w

    #### 复制文字

    	 esc + w

    #### 黏贴文字

    	 C + w

    #### 删除某行文字

    	 C + k
